# Manic NDS/3DS Skins

A vibrant NDS and 3DS skin catalog for [Manic EMU](https://github.com/Manic-EMU/WizardiOS) / Delta, built on top of the base skin art from [Manic-EMU/RetroSkinner](https://github.com/Manic-EMU/RetroSkinner). Every skin carries a full iPad (`standard`, portrait + landscape) representation alongside the original iPhone `standard`/`edgeToEdge` representations, so the same file works across devices.

## What's in here

- **`Skins/NDS Colorways/`** -- 108 mechanical recolors across 4 base families (`NotWhiteDS`, `NotWhiteDS FLEX`, `NotBlack3DS`, `NotBlack3DS FLEX`) x 27 vibrant colorways + rainbow. Generated via HSV hue-shift recoloring that preserves the original art's shading, gloss, and ambient occlusion -- not a flat tint.
- **`Skins/Premium NDS/`** -- 25 named/special-edition DS skins (Zelda: Spirit Tracks with a functional custom microphone/blow-whistle button, Zelda: Phantom Hourglass, Pokemon Black/White Edition, Mario Kart DS + 6 character variants, and more original concepts), each with hand-placed badge art positioned in the shell's real open space.
- **`Skins/Premium 3DS/`** -- 17 named/special-edition 3DS skins (Pokemon Sun & Moon, Pokemon Ultra Sun & Ultra Moon, and 15 original concepts).

## Installing on iOS

Download a skin file (`.manicskin`/`.deltaskin`) and open it with the Manic EMU / Delta app -- via AirDrop, the Files app share sheet, or the app's own skin-import screen.

## Releases

Each [release](../../releases) ships two zip assets:

- **Full Collection** -- every skin in the repo, latest version.
- **1G1R Curated Pack** -- borrowing the ROM-collecting "1 Game 1 ROM" idea: the 108 mechanical colorways are 4 families x 27 near-duplicate color variants, so the curated pack keeps exactly one flagship color (Rainbow) per family instead of all 27, plus the full premium set (all 42 premium skins are unique designs, not variants of each other, so none are dropped). Smaller download, same variety of *designs*.
