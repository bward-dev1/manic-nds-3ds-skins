# Premium 3DS Skins

17 named/special-edition 3DS skins built on `3DS.manicskin`, with the same hand-placed badge treatment as the Premium NDS set (hinge-gap placement, verified clear of the shell's L/LR shoulder labels and speaker/mic grille dots).

Includes Pokemon Sun & Moon, Pokemon Ultra Sun & Ultra Moon, and 15 original concepts (Cosmic Wormhole, Crimson Tactics, Haunted Mansion, Power Suit Hunter, Dream Rainbow, and others).
