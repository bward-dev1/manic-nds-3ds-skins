# NDS Colorways

108 mechanical recolors of the DS/3DS base skins, generated by HSV hue-shifting the original artwork (preserving shading, gloss, and AO -- not flat-tinted) across every vibrant colorway plus rainbow, in 27 colors x 4 base families:

- `NotWhiteDS/` -- recolors of `iswhiteDS.manicskin`
- `NotWhiteDS FLEX/` -- recolors of `DS_FLEX.manicskin`
- `NotBlack3DS/` -- recolors of `3DS.manicskin`
- `NotBlack3DS FLEX/` -- recolors of `3DS_FLEX.manicskin`

Each carries the same iPad standard (portrait + landscape) and iPhone standard/edgeToEdge representations as its base file.
