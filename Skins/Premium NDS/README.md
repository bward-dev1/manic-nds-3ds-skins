# Premium NDS Skins

25 named/special-edition DS skins built on `iswhiteDS.manicskin`, each with hand-placed decorative badge art positioned in the real open space of the shell (the hinge gap between the two screens, clear of the shoulder-button pills and the "MANIC EMU" wordmark) rather than a fixed copy-pasted spot.

Includes Zelda: Spirit Tracks (with a functional custom "microphone" input button styled as a train whistle, for the blow/mic mechanics), Zelda: Phantom Hourglass, Pokemon Black/White Edition, Mario Kart DS (master + 6 character variants), Rainbow Pride, Moonlight, Liquid Fusion, Bubbly Blob Rainbow, and other original concepts.
